โครงสร้างโปรเจค 
จะมีโฟลเดอหลักๆอยู่ด้วยกัน3โฟลเดอร์

1
node_modules - ใช้เก็บโมดูลหรือไลบรารี่ที่ทำงานภายในโปรเจคของเรา
2
public - เก็บไฟล์ต่างๆที่เป็นสาธารณะต่างๆเช่น รูปภาพ และ index.html
3
src - เก็บ componentโครงสร้างรวมๆของแอพพลิเคชั่น

ไฟล์สำคัญ
package.json - เ็บข้อมูลทุกอย่างและแพ็คเกตที่เราลงไว้ใช้งาน

1
/public/index.html
<div id="root"></div>
เป็นไฟล์ที่แสดงผลข้อมูลบนเว็บเบราเซอร์ จะรันผ่านไฟล์ index.html 
โดยเราจะให้ความสำคัญกับ <body> ตรงส่วนของ <div id="root"></div> ที่อยู่ใน index.html 
เพราะมันไปอ้างอิงกับcomponentที่เราสร้างขึ้นมา
2
/src/app.js
function App() {}
เป็นไฟล์หลักในการรันแอพขึ้นมาโดยนำส่วนประกอบต่างๆมาประกอบแล้วนำไปแสดงผลในบราวเซอร์ เป็นหน้าตา
ซึ่งapp component จะแสดงผลในบราวเซอร์ตรงพื้นที่ของ <div id="root"></div> ใน index.html
ถ้าต้องการปรับหน้าตาเว็บต้องมาแก้ ใน app.js ส่วนcomponentนั้นๆ
3
/src/index.js 
const root = ReactDOM.createRoot(document.getElementById('root'));
เป็นไฟล์ที่เชื่อมการทำงานระหว่างapp.jsและindex.html component ใน app.js เป็นตัวเรียกใช้เป็นตัวอ้างอิงค์
โดยจะใช้พวกคำสั่งแนว reactdom ในการ render
