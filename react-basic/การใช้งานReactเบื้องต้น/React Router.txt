React Router
การนำทางไปหน้าอื่นๆเช่นหน้า 1 2 3 4 โดยใช้Routeนำทางให้เข้าถึง component ต่างๆ โดยมี
Route เป็นลิงค์ในการเปิด
component เป็นหน้าของแอพแต่ละหน้า

การใช้คือ
ติดตั้ง npm install react-router-dom

แล้วทำการสร้างเราเตอหรือเราเพื่อกำหนดเส้นทางไปให้เข้าถึงคอมโพแนนของการกำหนดพาท
แล้วสร้างสวิชในการเปลี่ยนเส้นทางให้เข้าถึงคอมโพแนน
แล้วทำการกำหนดลิงค์เพื่อเชื่อมโยงพาทกับคอมโพแนน
