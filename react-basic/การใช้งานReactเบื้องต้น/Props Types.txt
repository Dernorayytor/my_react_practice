Props Types (การกำหนดหรือประกาศรูปแบบของprops)
เป็นการกำหนดหรือประกาศรูปแบบหรือชนิดของprops ที่ส่งเข้าไปทำงานในcomponent เช่นกำหนดชนิดข้อมูลของ Props 
หรือบังคับให้ต้องกำหนดค่า props ทุกๆครั้งที่มีการเรียกใช้งาน Component
รูปแบบ 

ชื่อComponent.popsTypes={
    ชื่อPropertie: รูปแบบของPropertie
}

https://www.npmjs.com/package/prop-types ใช้สำหรับเอาไว้ลง popsTypes เพื่อดึง .popsTypes มาใช้งาน
